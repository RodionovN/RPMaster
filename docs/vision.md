# RPMaster - Техническое видение проекта

## 1. Технологии

### Обязательные технологии

- **React Native** - кроссплатформенный фреймворк для iOS и Android
- **TypeScript** - типизация для надежности кода

### Предлагаемый минимальный стек

**Управление состоянием:**
- **React Context API** - простое встроенное решение, без дополнительных зависимостей

**Навигация:**
- **React Navigation** (v6) - стандарт для React Native

**Хранение данных:**
- **AsyncStorage** - встроенное решение React Native, простое и достаточное

**Работа с файлами:**
- **react-native-document-picker** - для выбора JSON файлов при импорте

**Валидация JSON:**
- Простая валидация вручную (проверка обязательных полей) - без дополнительных библиотек

**Генерация случайных чисел:**
- **Math.random()** - встроенная функция JavaScript

**Стилизация:**
- **StyleSheet** из React Native - без UI библиотек
